# radiowitness.io

HTLM CSS & JavaScript source for the [Radio Witness](https://radiowitness.io) web app.

## Setup
```
$ npm install
```

## Build
```
$ npm run build 
```

## Testing
```
$ vim js/config/config.js
$ npm run dev
```
